# jh-utils
这是一个工具方法库

* 此仓库包含一些常用的工具方法
* 组件库和文档工程化，用 [father](https://github.com/umijs/father) 和 [docz](https://www.docz.site/) 打包构建
* 文档发布在 [gh-pages](https://github.com/laixstx/jh-utils/tree/gh-pages) 分支
