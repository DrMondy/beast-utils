# byte2KOM
将 B 转换成 K 或者 M

### 参数

```typescript
/**
 * 将 B 转换成 K 或者 M
 * @param size
 * @private
 */
const byte2KOM = (size) => string;
```
### 实例

```javascript
let size  = '1024'
let byte = byte2KOM(size);

/**
* 输出：
* byte = '1.00K'
*/

```
