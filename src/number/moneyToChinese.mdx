# moneyToChinese
 数字金额大写转换(可以处理整数,小数,负数),注意，需要在浏览器缓存中设置 locale 语言 umi_locale:'zh-CN'

 否则 返回 空字符串

### 参数

```typescript
/**
 * 数字金额大写转换(可以处理整数,小数,负数)
 * @param money 数字金额
 * @returns {string}
 */
const moneyToChinese = (money) => money;
```

### 实例

```javascript
let val  = 566000
let money = moneyToChinese(val);

/**
* 输出：
* money = '伍陆万陆仟元'
*/

```
