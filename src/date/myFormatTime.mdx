# myFormatTime
传入参数 时间戳 和 format 属性，该方法会根据format属性 输出相应的日期 格式


### 参数

```typescript
/**
 * 将 时间戳 转成 日期
 * @param timestamp
 * @param format ('YYYY-MM-DD HH:mm:ss')
 * @returns {string}
 */
const myFormatTime = (timestamp,format='') => 'YYYY-MM-DD HH:mm:ss';
```

### 实例

```javascript

let date = myFormatTime('1577808000','YYYY-MM-DD');

/**
*输出：
* date = '2020-01-01';
*/

```
